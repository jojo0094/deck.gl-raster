# Introduction

deck.gl-raster is a collection of deck.gl layers and utilities for visualizing raster data on the web.

## Packages

| Package                             | Description                          |
| ----------------------------------- | ------------------------------------ |
| `@developmentseed/deck.gl-raster`   | Core raster layer for deck.gl        |
| `@developmentseed/deck.gl-geotiff`  | GeoTIFF / COG visualization layers   |
| `@developmentseed/deck.gl-zarr`     | Zarr visualization layers            |
| `@developmentseed/raster-reproject` | Client-side raster reprojection      |
| `@developmentseed/geotiff`          | Low-level GeoTIFF decoding utilities |
| `@developmentseed/morecantile`      | Tile matrix set utilities            |
| `@developmentseed/affine`           | Affine transform utilities           |
| `@developmentseed/epsg`             | EPSG projection definitions          |
